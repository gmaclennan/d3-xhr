[d3.xhr](https://github.com/mbostock/d3/wiki/Requests#xhr) packaged as an npm module for use with browserify.

### Usage

#### Browserify

```javascript
var xhr = require('d3-xhr');
xhr(url, mimeType, callback);
```

#### In the browser

```html
<script src="dist/d3_xhr.min.js"></script>

<script>
xhr(url, mimeType, callback);
</script>
```
